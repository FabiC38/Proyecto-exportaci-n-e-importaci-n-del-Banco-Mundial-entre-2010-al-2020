# Proyecto-exportaci-n-e-importaci-n-del-Banco-Mundial-entre-2010-al-2020
Proyecto de Power BI para el curso de Analista de Datos de Coderhouse 
El objetivo del proyecto es analizar las diversas importaciones y
exportaciones que tuvieron los diferentes países entre los años
2010 al 2020, para observar los cambios que se presenten como
también los porcentajes de los diversos servicios comerciales y
mercancías.
Además, a esto se le suma los cambios de las exportaciones e
importaciones de Argentina entre esos años.
